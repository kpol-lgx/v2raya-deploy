# v2raya-deploy
deploy v2raya and v2ray quickly in Linux

# platform
Linux, amd64 and aarch64

# install
```shell
git clone https://github.com/kpol-lgx/v2raya-deploy.git
cd v2raya-deploy
source v2raya-deploy.sh
```

# use v2raya
```shell
sudo v2raya
```

In chrome, firefox or edge, open `127.0.0.1:2017`
